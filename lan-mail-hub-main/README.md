# LAN Mail Hub

## Project Overview

LAN Mail Hub is a modern web application built using React, TypeScript, and Tailwind CSS.  
It is designed with a clean UI and scalable architecture for future expansion.

---

## Tech Stack

This project is built with:

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
- Supabase (for backend services)

---

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have the following installed:

- Node.js (LTS recommended)
- npm (comes with Node.js)

Check installation:
node -v
npm -v
